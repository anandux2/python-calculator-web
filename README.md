# Python Calculator Project

A comprehensive calculator project featuring both Command Line (CLI) and Graphical (GUI) interfaces.

## Features
- **Arithmetic Operations:** Addition, Subtraction, Multiplication, Division.
- **Advanced Operations:** Exponentiation, Modulus.
- **Error Handling:** Handles division by zero and invalid inputs gracefully.
- **CLI Interface:** Interactive loop-based command line version.
- **GUI Interface:** Modern dark-themed desktop application built with Tkinter.

## How to Run

### 1. CLI Version
Open your terminal and run:
```bash
python calculator_cli.py
```

### 2. GUI Version
Open your terminal and run:
```bash
python calculator_gui.py
```

## Setup Instructions
1. Ensure you have Python installed.
2. Clone or navigate to this directory.
3. No external dependencies are required (uses standard libraries).
